The Challenge

Retro Grid Constraints

Retro games run at ~256x240 resolution. Massively upscaling for modern 4K displays destroys visual intent if done incorrectly.

Failure of SOTA

Models like SwinIR or SRGAN interpret sharp edges as "noise," creating a blurry "oil painting" effect on pixel art.
